This repository contains a set of Bash scripts developed to explore open-source concepts through practical interaction with a Linux system, with a focus on Python as an open-source technology.

The project combines theoretical understanding of open-source software with hands-on implementation using shell scripting.

Features
System Identity Report
Displays Linux distribution, kernel version, user details, uptime, and licensing info
Python Package Inspector
Checks installation status and details of packages, highlighting open-source philosophy
Disk & Permission Auditor
Analyzes important system directories for permissions and storage usage
Log File Analyzer
Searches and counts keyword occurrences in log files and shows recent matches
Open Source Manifesto Generator
Generates a personalized open-source philosophy file through user input
Objectives
Understand open-source software concepts
Study Python as an open-source language
Explore Linux system interaction
Gain practical experience with shell scripting
Technologies Used
Bash scripting
Linux utilities (grep, awk, du, dpkg, apt)
Ubuntu/Linux environment
Learning Outcome

This project strengthens understanding of open-source principles while building practical skills in Linux and shell scripting, bridging the gap between theory and real-world implementation.
